# calculadora.frontend
Configuración del Proyecto:

1.-Se crea un archivo de nombre calculadora.html en donde se crean los principales
elementos graficos del proyecto, como los botones de la calculadora y a su vez se 
enlaza con las dependencias bootstrap, javaScrip y css.
2.-En una archivo de nombre calculadora.js se realizan las funciones necesarias para 
realizar las operaciones aritmeticas de la calculadora y mostrar en la display los 
resultados obtenidos, se utiliza la libreria display de JavaScript.
3.-Se crea un archivo de nombre calculadora.css en donde configuran los estilos, margenes,
fondos, alineación y la forma de los botones. 

Como Ejecutar la Calculadora:
1.-La interfaz grafica del proyecto es muy amigable, al abrir el sitio se observa
una calculadora con los elementos para realizar las operaciones aritmeticas basicas, asi 
como un boton para borrar el ultimo elemento introducido y otro boton para borrar en su 
totalidad los datos que se muestran en la display.
